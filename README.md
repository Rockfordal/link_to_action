# LinkToAction

Set of helpers: link_to_new, link_to_edit, link_to_destroy

## Installation

Add this line to your application's Gemfile:

    gem 'link_to_action'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install link_to_action

## Usage

link_to_{action} object, options
